 ## 📘 Curso de Front-end com React – IFMG

Este repositório tem como objetivo armazenar e organizar os arquivos desenvolvidos durante o **curso de Front-end com React** realizado pelo **IFMG**.

O repositório é **público** e serve tanto para acompanhamento da evolução do aprendizado quanto como portfólio dos projetos e exercícios desenvolvidos.

---

## 📂 Estrutura do Repositório

O projeto está organizado em duas pastas principais:

---

### 📌 1. AtividadesEntregar

Esta pasta contém as **atividades propostas durante o curso**, organizadas em subpastas.

- Cada subpasta representa uma atividade
- Foco em prática e fixação do conteúdo
- Pode incluir:
  - Exercícios
  - Desafios
  - Projetos com React

---

### 📌 2. AulasPresenciais

Esta pasta contém os **conteúdos desenvolvidos durante as aulas presenciais**, organizados por módulos.

Cada módulo possui um tema específico do curso:

#### 📦 Módulos do Curso

- **Módulo 1 – Lógica de Programação com JavaScript**  
  Fundamentos da programação, variáveis, estruturas condicionais, loops e funções.

- **Módulo 2 – Noções de UX e UI**  
  Princípios de usabilidade, design de interfaces e experiência do usuário.

- **Módulo 3 – Programação e Orientação a Objetos**  
  Conceitos de classes, objetos, encapsulamento e boas práticas de código.

- **Módulo 4 – HTML + CSS**  
  Estruturação de páginas e estilização com foco em layout e responsividade.

- **Módulo 5 – Desenvolvimento com React**  
  Criação de aplicações SPA, componentes, estados, props e hooks.

- **Módulo 6 – Web Services**  
  Consumo de APIs, requisições HTTP e integração com backend.

- **Módulo 7 – Soft Skills**  
  Comunicação, trabalho em equipe e desenvolvimento profissional.

- **Módulo 8 – Git**  
  Controle de versão, versionamento de código e uso do Git/GitHub.
